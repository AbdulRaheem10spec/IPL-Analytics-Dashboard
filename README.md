# IPL Analytics Dashboard

Interactive IPL analytics dashboard built using Python, Streamlit, Pandas, Plotly, and Machine Learning with real IPL ball-by-ball data.

---

## Features

- Smart IPL Points Table
- Playoff Prediction
- Venue Intelligence
- Team Head-to-Head Analysis
- Player Comparison Hub
- Player Form Tracker
- Match Momentum Timeline
- Live Win Probability Predictor
- Batter vs Bowler Analysis
- ML-based Match Prediction
- Orange & Purple Cap Prediction

---

## Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- Scikit-learn

---

## Dataset

Real IPL ball-by-ball dataset from Cricsheet.

Dataset Source:
https://cricsheet.org/

---

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
